# Advent of Code

This repo will hold the code for the adventofcode challenges for each year. Programs will be ranked based on performance based on the table below.

## Performance ranks
| Time | Icon |
| :--: | :--: |
| <1ms | :blue_heart: |
| 1-5ms | :green_heart: |
| 5-10ms | :yellow_heart: |
| 10+ms | :heart: |
